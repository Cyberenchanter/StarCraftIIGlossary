# StarCraft II Glossary Generator

Scripts used to generate ini glossary files for StarCraft II arcade projects' localization. Originally created for project Nexus Co-op. See [config.py](config.py) for configurations.

## Usage

```
git clone github.com/cyberenchanter/StarCraftIIGlossary.git
cd StarCraftIIGlossary
python generate_glossary.py
```
